    Symbols listed on the webpage


   
>

   
The > entity represents the greater-than sign(>) .It is used
      to display the greater-than symbol in HTML. Example:5>3
   

   
&;

   
The & entity represents the ampersand (&). It is used to display
      the ampersand symbol in HTML documents. Example:Smith&Co.
   

   
"

   
The " entity represents the double quotation mark("Hi I Am Syeda") .It  
      is used to display double quotes in HTML documents. Example:He said,"
      Hello,world! "
   

   
'

   
The ' entity represents the single quotation mark or apostrophe
      ('MIRHAKAZMI77').It is used to display single quotes or apostrophe
      in HTML documents.Example:It's a beautiful day!
   

   
 
 
   
The   entity represents a non-breaking space.It is used to 
      create a space between words or characters that will not to be 
      break by a line break.Example:Hello, world!(prevents lined
      break between "Hello", and "World!")

The $ symbol represents the dollar sign ($). It is not an HTML  
      entity, but rather a literal character.Example:The price 
      is $10.99
