THIS IS A AUDIO TAG

Audio definition

The HTML <audio> tag is used to embed sound content in a web page, such as music or other audio streams. It is a self-contained tag and does not require a closing tag.

The <audio> tag has two required attributes
Syntax

The <audio> tag has two required attributes:

  • src: Specifies the path to the audio file. This can be a relative or absolute URL.
  • controls: When this attribute is present, it adds audio controls, such as play, pause, and volume, to the audio player.

Additionally, the <audio> tag can contain multiple <source> elements to provide different audio formats for better browser compatibility. The browser will use the first format it supports.

Attributes of <audio> tag

Attribute Description
autoplay This attribute specifies that the audio should start playing as soon as it is ready, without requiring user interaction.
controls This attribute adds audio controls, such as play, pause, and volume, to the audio player.
loop This attribute specifies that the audio should start over again, every time it is finished.
muted This attribute specifies that the audio output should be muted (no sound).
preload This attribute specifies if and how the author thinks that the audio should be loaded when the page loads. It can take three values: "auto", "metadata", and "none".

Example of <audio> tag

Here is an example of how to use the <audio> tag in an HTML document:

<audio controls>
<source src""audiofile.mp3.mp.mp3" type="audio/mpeg">
</audio>
